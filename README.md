# 💪 APEX — High Performance Fitness Hub

A modern, premium fitness landing page built with **HTML5**, **Tailwind CSS**, and **Vanilla JavaScript**. Designed with a futuristic aesthetic, smooth animations, and conversion-focused sections, APEX delivers a high-end user experience for gyms, fitness centers, and personal training brands.

## 🚀 Features

- Premium splash screen loader
- Responsive glassmorphism navigation
- Animated hero section
- Floating gradient background effects
- Interactive feature cards
- Statistics & impact section
- Specialized training programs
- Transformation showcase
- Modern pricing plans
- Lead generation form
- SVG section dividers
- Smooth animations
- Mobile-first responsive design

## 🛠️ Built With

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Google Fonts
- SVG Graphics
